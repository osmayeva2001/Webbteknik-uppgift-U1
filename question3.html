b. Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Både Id och class är attributer som man använder i CSS. Id använder man för att identifiera element på en sida, medan en class används för att klassificera elementen. Med id kan man t.ex styla med CSS och det får bara finnas ett element med samma Id på hela sidan. Med Id kan man till exempel identifiera huvudmenyn för att skilja ut den från t.ex sidomenyer.

Class är en attribut som man klassificerar elementen med. Till skillnad från en Id, kan man använda element med samma klass flera gånger på samma sida.
